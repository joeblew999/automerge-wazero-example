How it works:

  • Get Cursor: Create a cursor at a specific position in the text
  • Lookup Cursor: Find the current position of a previously saved cursor
  • Stable Positions: Cursors adjust automatically when text is edited
  • Demo: Shows cursor tracking across edits:
    1. Set text to "Hello World"
    2. Get cursor at position 6 (after "Hello ")
    3. Insert "Hi " at the beginning
    4. Lookup cursor ‚Üí Position updates from 6 to 9

API Endpoints:

GET /api/cursor?path=ROOT.content&index=5 - Get cursor
POST /api/cursor/lookup {path, cursor} - Lookup position
